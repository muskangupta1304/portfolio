# Muskan Gupta 3D Portfolio

A premium, interactive 3‑D portfolio website built with **HTML**, **CSS**, and **Vanilla JavaScript**. It showcases:

- A Three.js animated background
- GSAP scroll‑triggered entrance animations
- Custom cursor with particle trail
- Typing animation for the subtitle
- Theme‑switcher with multiple color palettes
- Animated counters, quote section, and more

## Live Demo
https://muskangupta1304.github.io/portfolio

## Running locally
```powershell
# Clone the repo
git clone https://github.com/muskangupta1304/portfolio.git
cd portfolio
# Serve with any static server (Python example)
python -m http.server 8000
```
Open `http://localhost:8000` in your browser.

## Tech Stack
- **Three.js** – 3‑D background
- **GSAP & ScrollTrigger** – smooth scroll animations
- **Font Awesome** – icons
- **CSS Glassmorphism** – modern UI
- **Pure JavaScript** – no frameworks required

## Features
- Theme switcher (palette icon) – choose from 6 color schemes
- Scroll progress bar at the top
- Animated navigation with active‑section highlighting
- Counters that animate on scroll
- Personal quote, currently‑learning tags, language proficiency bars
- Resume download button
